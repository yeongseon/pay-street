# PayStreet Asset Guide

## 1. Overview

이 문서는 템플릿 영상에 사용되는 시각 자산 규칙을 정의한다.

---

## 2. Asset Categories

- backgrounds
- silhouettes
- interviewer assets
- overlays
- logos
- icons

---

## 3. Global Rules

- 모든 자산은 9:16 기준
- 브랜드 색상 일관성 유지
- 과도하게 복잡한 배경 금지
- 인터뷰이 자산은 익명성 유지

---

## 4. File Naming

예시

```text
bg_street_01.mp4
silhouette_person_01.png
overlay_salary_card_v1.png
logo_paystreet_white.png
```

---

## 5. Asset Quality

- 배경 영상은 loop 가능해야 함
- PNG 자산은 alpha 채널 유지
- 텍스트가 많은 자산은 피함
- subtitle 영역과 시각 충돌 없어야 함
